# 🎨 ArtistLife – Full Stack Art Sharing Platform

## 📌 Project Overview

**ArtistLife** is a full-stack web application where users can upload, view, and manage artworks.
It allows artists to showcase their creativity and explore artworks uploaded by others.

---

## 🚀 Features

* 🔐 User Authentication (Login System)
* 🖼️ Image Upload Functionality
* 🖼️ Gallery to View Uploaded Artworks
* 👤 User Profile Page
* 📊 Dashboard Overview
* ❤️ Like System (Optional Feature)
* 🌐 REST API Integration

---

## ⚙️ How to Run the Project

### 🔹 Backend (.NET API)

1. Navigate to backend folder:

```
cd backend
```

2. Run the API:

```
dotnet run
```

3. Open Swagger:

```
https://localhost:<port>/swagger
```

---

### 🔹 Frontend (Angular)

1. Navigate to frontend folder:

```
cd frontend
```

2. Install dependencies:

```
npm install
```

3. Run Angular app:

```
ng serve
```

4. Open browser:

```
http://localhost:4200
```
